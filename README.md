# obsidian-ly
create obsidian daily, weekly, monthly, quarterly, yearly base of notes
Usage: ./obsidian-ly.sh <path to create vault> <dirname of periodic subdirs + notes>
```
./obsidian-ly.sh ./vault base # will create:
.
 |-vault
   |-base
   |---daily
   |---weekly
   |---monthly
   |---quarterly
   |---yearly
   |-templates
   |---Default Template.md
   |---filename_template.md
   |---.... etc ....
```
